Simulasi Algoritma

Coba jalankan algoritma "Login ke akun" berikut. Tekan tombol Jalankan untuk melihat tiap langkah berjalan satu per satu.

📥 Mulai
1. Buka halaman login
2. Masukkan username
3. Masukkan password
4. Klik tombol Login
5. Cek: apakah username & password benar?
➜ Ya → Masuk ke halaman utama
➜ Tidak → Tampilkan pesan error
📤 Selesai

Langkah ke-5 adalah percabangan (kondisi). Algoritma akan memilih jalur berbeda tergantung apakah username dan password benar atau tidak.
